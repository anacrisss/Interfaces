Elementos básicos:

El formato de archivo SVG es una herramienta que se utiliza habitualmente para mostrar gráficos, diagramas e ilustraciones bidimensionales en los sitios web. Se trata de un formato de imagen vectorial por lo que se puede ampliar y reducir sin perder resolución.

Tiene integración muy sencilla con HTML a través de elementos tipo canvas <svg>

Los elementos más sencillos van a ser:

  • Rectángulos: <rect>
  • Círculos: <circle>
  • Elipses: <ellipse>
  • Líneas <line>
  • Multilíneas: <polyline>
  • Paths: <path>
  • Polígonos: <polygon>
