Patrones

Mediante el elemento pattern dentro de un defs podemos diseñar un patrón que se va a repetir y que podemos emplear para rellenar el fondo de otros elmentos.


  • El atributo x e y marca donde comienza el primer tile del patrón.
  • El atributo width y height controlan el tamaño de la tile que se repite.
  • El atributo patternUnits="userSpaceOnUse" define las unidades del patrón y esta es la opción por defecto que emplea px
  • Dentro del pattern definimos los elementos que van a estar dentro de la tile.
