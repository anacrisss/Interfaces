Gradientes

Los gradientes son la combinación de uno o varios colores ordenados de forma lineal en la que se realiza una transición gradual, suave y progresiva. Podemos emplearlos para rellenar el interior de otros elementos definidos en svg.

Tenemos gradientes básicos de dos tipos:

Lineales

Debemos definirlo dentro de un elemento defs mediante la instrucción linearGradient.

Debemos tener en cuenta que:

  • El atributo offset define los pasos del gradiente
  • El atributo stop-color define el color de cada paso
  • Podemos tener dos o más pasos en la definición de los colores de nuestro gradiente
  • Debemos asociar un id al elemento linearGradient que luego referiremos desde el atributo fill del elemento que queramos rellenar con dicho gradiente tal que fill=url(#MyGradId)

Radiales

Debemos definirlo dentro de un elemento defs mediante la instrucción radialGradient. Este elemento admite atributos parecidos a la elipse para definir el comportamiento del gradiente.

  • Los atributos cx and cy definen la posición x e y del círculo final del gradiente
  • Los atributos fx and fy definen la posición x e y del círculo de inicio del gradiente.
  • El atributo r define el radio del círculo final del gradiente
  • Los colores del gradiente se definen mediante dos o más elementos <stop>
  • El atributo offset define los pasos del gradiente
  • El atributo stop-color define el color de cada paso
  • Debemos asociar un id al elemento linearGradient que luego referiremos desde el atributo fill del elemento que queramos rellenar con dicho gradiente tal que fill=url(#MyGradId)
