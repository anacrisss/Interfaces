Práctica de diseño de botones en CSS puro

Ejercicio 1: Recopila las propiedades más relevantes para el diseño de botones custom en CSS, explicando brevemente la función de cada una en el diseño
A. Fondo y color: 
    background-color: Define el color de fondo del botón.
    Ejemplo: background-color: #4CAF50; (verde).

    background-image: Permite usar gradientes o imágenes como fondo.
    Ejemplo: background-image: linear-gradient(to bottom, #4CAF50, #2E7D32);
    Ejemplo con imagen: linear-gradient(to bottom right, rgba(0, 128, 0, 0.9),
            rgba(255, 255, 0, 0.7)), url("logo_tech.png");

    color: Establece el color del texto en el botón.
    Ejemplo: color: white;
B. Bordes y esquinas: 
    border: Establece el tamaño, tipo y color del borde.
    Ejemplo: border: 2px solid #4CAF50;

    border-radius: Redondea las esquinas del botón.
    Ejemplo: border-radius: 10px; (botón con esquinas redondeadas).

    box-shadow: Añade sombra al botón para darle profundidad.
    Ejemplo: box-shadow: 4px 4px 6px rgba(0, 0, 0, 0.3);
C. Tamaño-espaciado y Contenido-alineación: 
    padding: Controla el espacio interno entre el contenido (texto o íconos) y los bordes del botón.
    Ejemplo: padding: 10px 20px;

    margin: Define el espacio externo alrededor del botón.
    Ejemplo: margin: 10px;
    
    width y height: Especifica las dimensiones del botón.
    Ejemplo: width: 150px; height: 50px;
    ------
    text-align: Alinea el texto dentro del botón (izquierda, derecha o centrado).
    Ejemplo: text-align: center;

    display: Controla cómo se muestra el botón.
    Ejemplo: display: inline-block; para que el botón no ocupe toda la línea.

    vertical-align: Ajusta la posición vertical del contenido.
D. Estilo al interactuar (estado del botón): 

    hover (:hover): Cambia el estilo cuando el cursor pasa por encima.
    Ejemplo:
        button:hover {
        background-color: #45a049;
        transform: scale(1.05); /* Efecto de agrandamiento */
    }

   active  (:active): Cambia el estilo cuando el botón está presionado.
    Ejemplo:
       button:active {
        background-color: #2e7d32;
        box-shadow: inset 2px 2px 5px rgba(0, 0, 0, 0.5);
    }

   focus  (:focus): Define el estilo al recibir foco, útil para accesibilidad.
    Ejemplo:
        button:focus {
        outline: 2px solid #4CAF50;
    };
E. Animaciones y transiciones: 

    transition: Crea efectos suaves para cambios en propiedades como color, tamaño o sombra.
    Ejemplo:
    
    button {
        transition:  all 0.3s ease;
    }
    
    transform: Permite rotar, escalar o mover el botón.
    Ejemplo:
    
    button:hover {
        transform: scale(1.1); /* Aumenta ligeramente el tamaño */
    }
F. Efectos avanzados: 
    filter: Agrega efectos como brillo o desenfoque.
    Ejemplo: filter: brightness(1.2); para hacer el botón más luminoso al pasar el cursor.

    clip-path: Permite crear formas personalizadas para el botón.
    Ejemplo: clip-path: polygon(0 0, 100% 0, 100% 100%, 0 90%); (botón con una forma trapezoidal).
